1.  **Chatbot Core**: A chatbot where an admin can create a decision tree of questions and answers.
2.  **Payment Integrations**: Integrate Razorpay and Cashfree.
3.  **Admin Panel**:
    *   CRUD for the Q&A decision tree, including platform-specific questions and agent handoff toggles.
    *   Secure management of Payment Gateway credentials.
    *   A View Journey feature to visualize the Q&A flow.
    *   CRUD for managing agents, including assigning platforms and roles ('agent' or 'master agent').
4.  **Agent Live Chat**:
    *   Real-time WebSocket chat for agents to talk to end-users.
    *   Agents are assigned to specific platforms.
    *   Agents can see the user's full chat history.
    *   **Master Agents**: A special role that can view all chats (respecting platform restrictions) and re-assign/un-assign chats between other agents.
5.  **Refined Chat Flow**:
    *   Agent handoff is triggered contextually when a user expresses dissatisfaction (e.g., clicking No, connect to agent after a bot answer) or by typing keywords like talk to agent. The persistent Talk to Agent button has been removed.
6.  **Multilingual Support**:
    *   Users select their preferred language at the beginning of a chat session.
    *   Agents can manage a list of languages they know via their profile.
    *   The user's chosen language is visible to agents in the dashboard.
7.  **SDK & Documentation**:
    *   Provide a PHP SDK, a JavaScript SDK, a user guide, and a technical integration guide.
    *   The admin panel should have a dedicated page to display this documentation and provide download links.
